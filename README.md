Este projeto consiste no desenvolvimento de uma plataforma web personalizada, projetada para consolidar e exibir de forma dinâmica a minha trajetória académica e técnica. Mais do que um simples currículo estático, o sistema integra uma linha do tempo de projetos, detalhamento de competências tecnológicas e uma interface bilíngue (Português/Inglês).

### **Principal Valor e Benefício**
O valor central deste projeto reside na **construção de uma identidade digital profissional de alto impacto**. 

**Os principais benefícios incluem:**
* **Credibilidade Instantânea:** Centralização de projetos (com links diretos para o GitHub), experiências e formas de contacto numa interface moderna e responsiva.
* **Diferenciação Competitiva:** Atua como uma ferramenta de conversão no mercado de tecnologia, permitindo que recrutadores e parceiros visualizem, de forma prática e visual, a minha evolução técnica e capacidade de entrega.
* **Presença Global:** Através da arquitetura preparada para multilinguismo e hospedagem na nuvem, o portfólio expande o alcance profissional para além das fronteiras locais.

## 🛠 Tecnologias Utilizadas

As seguintes ferramentas, frameworks e bibliotecas foram utilizados na construção deste projeto. Recomenda-se o uso das versões listadas (ou superiores) para garantir a compatibilidade.

### 💻 Front-end

* **Framework/Biblioteca:** [Anime.js](https://animejs.com/) (v3.2.1+) para animações do sistema de carregamento e [GSAP](https://gsap.com/) para transições de interface.
* **Linguagem/Superset:** JavaScript (ES6+) para lógica de sliders, animações de scroll e manipulação dinâmica do DOM.
* **Estilização:** CSS3 moderno utilizando Variáveis (`:root`), Flexbox, Grid Layout e animações `@keyframes` personalizadas.
* **Ícones e Tipografia:** Font Awesome v6.4.0 para iconografia técnica; Google Fonts (fontes: *Tajawal*, *Fira Code* e *Orbitron*).
* **APIs do Navegador:** [Intersection Observer API](https://www.google.com/search?q=https://developer.mozilla.org/pt-BR/docs/Web/API/Intersection_Observer_API) para ativação de animações ao entrar na área de visualização.
